## Understanding the Flow

The improved preview system works like this:

1. When parameters change, the specific node's preview is updated first.
2. Then the main preview is updated to show the complete output.
3. When a node is selected, its preview (showing only the chain up to that node) is displayed.
4. All previews are processed in the correct order using a topological sort.

This ensures that previews always reflect the current state of the node system and update immediately when changes are made.
