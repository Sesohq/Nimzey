2. In NodePreviewContainer (NodePreviewContainer.tsx)
You already have this listener wired up (around NodePreviewContainer):

// NodePreviewContainer.tsx
useEffect(() => {
  const handleParamChange = (e: CustomEvent) => {
    if (e.detail && e.detail.nodeId) {
      // Calls into the worker via updateNodePreview
      window.updateNodePreview?.(e.detail.nodeId);
    }
  };

  window.addEventListener('node-param-changed', handleParamChange as EventListener);
  return () => {
    window.removeEventListener('node-param-changed', handleParamChange as EventListener);
  };
}, []);
No changes needed here—this is exactly the hook that will now fire on every slider move, triggering the offscreen-canvas render and posting back the new thumbnail.
