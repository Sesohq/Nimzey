🔧 1. Understanding the Filter: Gradient Noise
The Wiki breaks this down into:

Gradient Noise (includes Perlin noise, Simplex noise)

Value noise vs. Gradient noise: Gradient noise uses directional gradients at lattice points, interpolated across space.

⛓️ To Implement a Noise Node:
js
Copy
Edit
function generatePerlinNoise(width, height, options) {
  const {
    octaves = 4,
    persistence = 0.5,
    frequency = 0.01,
    seed = 42
  } = options;

  const noise = createNoise2D(seed); // Use a lib like `open-simplex-noise` or `perlin-noise`
  const data = new Float32Array(width * height);

  for (let y = 0; y < height; y++) {
    for (let x = 0; x < width; x++) {
      let value = 0;
      let amp = 1;
      let freq = frequency;
      for (let o = 0; o < octaves; o++) {
        value += noise(x * freq, y * freq) * amp;
        amp *= persistence;
        freq *= 2;
      }
      data[y * width + x] = value;
    }
  }
  return data;
}
Then you convert data[] to image pixels (grayscale, or RGB via color mapping).
